## migration

Plan and execute migrations safely.

### Process
1. Audit current usage
2. Migration plan with timeline
3. Add deprecation warnings
4. Run old and new in parallel
5. Test both paths
6. Remove old code
7. Update docs

### Types
- Dependency upgrades
- Framework migrations
- Language upgrades
- Database migrations
- API versioning
