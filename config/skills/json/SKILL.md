## json

Process structured data formats.

### JSON tips
- Use jq for shell processing
- Validate with JSON Schema
- Pretty-print for readability
- Never eval/JSON.parse untrusted data

### YAML vs TOML vs JSON
- YAML: complex configs
- TOML: simple configs (pyproject.toml)
- JSON: data exchange, APIs
- Pick the right format
