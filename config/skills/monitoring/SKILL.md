## monitoring

Set up effective monitoring.

### Three pillars
- Logs: structured event records
- Metrics: numerical measurements (CPU, latency, error rate)
- Traces: end-to-end request flow

### Key metrics
- RED: Rate, Errors, Duration
- USE: Utilization, Saturation, Errors
- SLO/SLI: Service Level Objectives
- Apdex: user satisfaction
