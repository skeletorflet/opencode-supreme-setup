## markdown

Write clean markdown.

### Structure
- H1: page title (one per doc)
- H2: major sections
- H3: subsections
- Blank lines between blocks

### Elements
- **bold** for emphasis
- code for inline code
- ` for code blocks with language
- > For blockquotes
- - For unordered lists
- 1. For ordered lists
- [text](url) for links
